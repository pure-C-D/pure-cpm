# PureCPM — Portable CPM Schedule Engine

A lightweight, browser-based Critical Path Method (CPM) scheduling tool built for construction project managers who need to update schedules from the field.

## Features

- **CPM Engine** — Forward/backward pass, critical path, total float
- **Mobile-First** — Card-based update view designed for phones and tablets
- **P6 Compatible** — Import/export CSV files for Primavera P6 round-trip workflow
- **Auto-Save** — Projects persist in your browser automatically
- **Multi-Project** — Manage multiple schedules from one app
- **Gantt Chart** — Visual schedule with data date line and actual vs. planned bars
- **All 4 Relationship Types** — FS, SS, FF, SF with lag support
- **5-Day Work Week** — Skips weekends automatically
- **Zero Install** — Single HTML file, runs in any browser

## Field Workflow

1. Export schedule from P6 as CSV
2. Import into PureCPM on your phone
3. Update % complete and actual dates from the field
4. Export P6-compatible CSV
5. Import back into P6

## Usage

Open `index.html` in any browser, or visit the hosted version at your GitHub Pages URL.
